# VMS - Voucher Management System


## Installation et lancement

1. Télécharger le projet
   Téléchargez le fichier compressé (ZIP) contenant l’application.

2. Extraire les fichiers
   Extrayez le contenu du fichier ZIP.

3. Lancer les applications
   Ouvrez les deux applications suivantes :

* VMS (application principale de gestion)
* ScanQrApp (application de scan en magasin)

Assurez-vous que les deux applications sont lancées pour un fonctionnement complet du système.

---

## Fonctionnalités principales

### VMS

* Création de bons d’achat
* Envoi des bons par email
* Gestion des utilisateurs et des rôles
* Suivi des bons émis

### ScanQrApp

* Scan des QR codes via webcam
* Vérification du bon d’achat
* Validation unique (un bon ne peut être utilisé qu’une seule fois)
* Contrôle du magasin associé au bon

---

## Base de données

Les applications sont connectées à une base de données distante hébergée sur AlwaysData.
Une connexion Internet est donc nécessaire pour utiliser les applications.
